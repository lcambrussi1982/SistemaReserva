Tutorial rápido

Professores

  1. Acesse a página Login e faça seu acesso.
  2. Você será redirecionado para Reservas com o grid semanal.
  3. Escolha a data, o tipo de dispositivo e clique em um horário disponível.
  4. Suas reservas do dia aparecem em Minhas reservas do dia.

Administradores

  1. Faça login com uma conta que tenha o campo role = "ADMIN" na coleção usuarios.
  2. Abra Dispositivos para ajustar o total de tablets e chromebooks.
  3. No futuro, a tela de Cadastros será usada para professores, disciplinas e turmas.

Tecnologia

Este sistema usa uma camada simples no navegador e salva os dados em um banco de dados na nuvem via Firebase Cloud Firestore.
